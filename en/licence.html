Appendix 1: Documentation

Free version

The free version includes the Seald desktop application, which can be installed in the usual way on each operating system.

The registration procedure requires access to the Internet and its mailbox.

Once installed, the application allows you to encrypt and decrypt documents. Just launch the application, a Seald icon will appear in the bar next to the time, on which you just have to click to make an encryption window appear. All you have to do then is follow the instructions.

To decrypt a document with the application (and therefore with a Seald account), simply double-click on it. It should be noted that it is for the moment impossible to decipher a document for a user of Seald without the installed application.

For someone who is not a user of Seald and therefore does not have a account, simply open the encrypted file in a browser compatible and follow the instructions.

On Windows, this version also includes the plugin for Microsoft Outlook (compatible versions in Annex 2).

Enterprise version

The Enterprise version includes everything that the free version includes, with tracking and control capabilities via the Administration Panel available for one or more administrators (on request) of your company, and via the application via reduced functionality (such as opening notifications).

The Administration Panel works in a compatible browser, just open the URL that will be indicated to you during the provisioning of the account with the identifiers and passwords that you will have chosen.
