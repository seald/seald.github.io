Toutes les réponses à vos questions

Quelles sont les types d'intégration au serveur ?

Il existe 3 types d'intégration:
- En installant directement l’agent «Seald for Server» sur le serveur applicatif
- En utilisant un WebService « Seald for Server » sur un serveur tiers (interne à l’organisation)
- Dans le cas de chiffrement d’emails uniquement, en utilisant un proxy SMTP.

Quels sont les besoins pour l'intégration agent "Seald for Server" ?

Les besoins sont les suivants. Tout d'abord, un système d’exploitation pouvant installer NodeJS 6 (ou supérieur) (Windows, Mac, Linux, zOS) ou Docker et une connectivité vers les serveurs Seald (HTTPS).

Comment se réalise l'intégration ?

Nous vous accompagnons dans le choix de l'approche et l'intégration de votre solution.
