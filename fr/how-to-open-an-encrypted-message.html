Comment ouvrir un email sécurisé ?

Si vous avez reçu un message chiffré d'une entreprise utilisant Seald, vous pouvez l'ouvrir de deux façons. Vous pouvez installer l'application Seald pour une ouverture transparente du message (télécharger Seald) ou vous pouvez vous authentifier pendant une courte période pour lire le message.

Avant de commencer

Qu'est-ce que Seald ?

Seald chiffre vos emails, pièces jointes et documents de manière transparente. Une solution de cryptographie facile à utiliser, conçue pour les entreprises, se différenciant des outils existants par sa convivialité et son accessibilité.

J'ai déjà un compte Seald. Dois-je suivre cette procédure pour afficher un message sécurisé ?

Non. Ce processus s’adresse uniquement aux personnes qui n’ont pas encore installé Seald. Ouvrez le document sur l'ordinateur avec votre compte existant pour afficher le message chiffré.

M'authentifier pour voir les messages chiffrés

Suivez ces instructions pour ouvrir un courrier électronique / document chiffré:

  1. Téléchargez ou ouvrez la pièce jointe avec l'extension .shtml dans votre message. Par exemple, une boîte de réception Outlook et Gmail contenant un message chiffré.

    Le fichier .shtml ne s'ouvre pas ? Cliquez ici.

    or

  2. Après avoir ouvert la pièce jointe, sélectionnez Ouvrir le document.

  3. Remplissez le champ avec votre email pour recevoir un code d'accès unique. L'adresse doit être celle que l'expéditeur a choisi au moment de chiffrer le document pour vous.

  4. Vous recevrez ensuite un code d'accès sur votre adresse e-mail.

  5. Entrez le code d'accès et confimez.

  6. Votre message est maintenant déchiffré. Bonne lecture.
