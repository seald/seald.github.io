Annexe 1: Documentation

Version gratuite

La version gratuite inclut l’application de bureau Seald, qu’il suffit d’installer selon la manière usuelle sur chaque système d’expoitation.

La procédure d’inscription nécessite accès à Internet et à sa boîte email.

Une fois installée, l’application permet de chiffrer et déchiffrer des documents. Il suffit de lancer l’application, une icone Seald va apparaitre dans la barre à côté de l’heure, sur laquelle il suffit de cliquer pour faire apparaitre une fenêtre de chiffrement. Il suffit alors de suivre les instructions.

Pour déchiffrer un document avec l’application (et donc avec un compte Seald), il suffit de double-cliquer dessus. Il est à noter qu’il est pour le moment impossible de déchiffrer un document pour un utilisateur de Seald sans l’application installée.

Pour quelqu’un qui n’est pas utilisateur de Seald et qui n’a donc pas de compte, il suffit d’ouvrir le fichier chiffré dans un navigateur compatible et de suivre les instructions.

Sur Windows, cette version inclut également le plugin pour Microsoft Outlook (versions compatibles en Annexe 2).

Version Entreprise

La version Entreprise inclut tout ce que la version gratuite inclut, avec des fonctionnalités de suivi et de contrôle a posteriori via le Tableau d’Administration disponible pour un ou plusieurs administrateurs (sur demande) de votre entreprise, et via l’application via des fonctionnalités réduites (telles que des notifications d’ouverture).

Le Tableau d’Administration fonctionne dans un navigateur compatible, il suffit de se connecter à l’URL qui vous sera indiquée lors du provisionnement du compte avec les identifiants et mots de passe que vous aurez choisis.

Chaque élément chiffré est référencé dans ce tableau avec ses destinataires, le nombre de lectures effectuées et un bouton révoquer pour chacun qu’il suffit d’utiliser pour empêcher toute nouvelle ouverture de l’élément.

Annexe 2: Configuration technique

Seald fonctionne sur:

  • Windows 7 SP1, 8, 10;
  • macOS 10.9 et supérieur;
  • Ubuntu 18.04 (fonctionnement non garanti);

Si le poste n’est pas aux paramètres par défaut (via politique de sécurité ou autre), cela peut mener à des dysfonctionnements.

Le format de fichier Seald permettant de déchiffrer sans installer l’application pour des destinataires n’ayant pas Seald:

  • Apple Safari 11;
  • Google Chrome 66;
  • Mozilla Firefox 60;
  • Microsoft Edge 17;
  • Microsoft Internet Explorer 11 (des dysfonctionnements peuvent survenir);

Le plugin pour Microsoft Outlook fonctionne sur Windows 7 SP1, 8 et 10 sur Microsoft 2013 et 2016, et fonctionne sur une version aux paramètres par défaut de Office 365 et/ou Exchange.

Le tableau d’administration est garanti de fonctionner sur Google Chrome 66, de légers dysfonctionnements peuvent survenir sur d’autres navigateurs.
